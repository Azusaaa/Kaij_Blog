## 技术栈

### 前端技术栈

- Vue
- Vue-Router
- Axios
- ElementUI
- Vue-cli

### 后端技术栈（由负责后端的同学提供

- Java Spring Boot框架
- Mysql数据库
- mybatis
- Spring Security
- 部分遵循restful接口
